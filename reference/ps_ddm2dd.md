# Degrees and Decimal Minutes to Decimal Degrees

Degrees and Decimal Minutes to Decimal Degrees

## Usage

``` r
ps_ddm2dd(x)
```

## Arguments

- x:

  A character vector.

## Value

A numeric vector.

## Examples

``` r
ps_ddm2dd(c("70 39.24", NA, "-56 39.24"))
#> [1]  70.654      NA -56.654
```
