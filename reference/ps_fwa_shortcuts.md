# FWA shortcuts

FWA shortcuts

## Usage

``` r
ps_fwa_shortcuts(
  gdb = "FWA_BC.gdb",
  dir = "~/Poisson/Files - Data/Data/spatial/fwa/gdb/"
)
```

## Arguments

- gdb:

  A character string vector indicating FWA geodatabases to extract layer
  shortcuts from.

- dir:

  A character string indicating path to directory holding fwa
  geodatabases.

## Value

A factor of the layer names within specified geodatabase.
