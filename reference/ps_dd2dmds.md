# Decimal Degrees to Degrees and Decimal minutes.

Decimal Degrees to Degrees and Decimal minutes.

## Usage

``` r
ps_dd2dmds(x)
```

## Arguments

- x:

  A numeric vector.

## Value

A character vector.

## Examples

``` r
ps_dd2dmds(c(70.654, NA, -56.654))
#> [1] "70 39 14.3999999999869"  NA                       
#> [3] "-56 39 14.4000000000125"
```
