# FWA layers

FWA layers

## Usage

``` r
ps_fwa_layers(
  gdb = "FWA_BC.gdb",
  dir = "~/Poisson/Files - Data/Data/spatial/fwa/gdb/"
)
```

## Arguments

- gdb:

  A character string vector indicating FWA geodatabases to extract
  layers from. The default should not have to be changed.

- dir:

  A character string indicating path to directory holding fwa
  geodatabases.

## Value

A factor of the layer names within specified geodatabase.
