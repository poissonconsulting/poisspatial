# Create sf rectangle

Create sf rectangle

## Usage

``` r
ps_sf_rectangle(x, crs)
```

## Arguments

- x:

  A bbox object (or numeric vector of length 4 indicating xmin, ymin,
  xmax, ymax).

- crs:

  coordinate reference system: integer with the EPSG code, or character
  with proj4string

## Value

sfc polygon.
