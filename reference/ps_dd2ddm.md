# Decimal Degrees to Degrees and Decimal Minutes.

Decimal Degrees to Degrees and Decimal Minutes.

## Usage

``` r
ps_dd2ddm(x)
```

## Arguments

- x:

  A numeric vector.

## Value

A character vector.

## Examples

``` r
ps_dd2ddm(c(70.654, NA, -56.654))
#> [1] "70 39.2399999999998"  NA                     "-56 39.2400000000002"
```
